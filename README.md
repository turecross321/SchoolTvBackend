# SchoolTvBackend
API for [SchoolTvFrontend](https://github.com/turecross321/SchoolTvFrontend) - a school project where the goal was to create an application for our program's TV that shows relevant information for students. More information about the final product including photos can be found in the frontend's repository.

This project is written in C# using ASP.NET, and it serves the following data
- The school's logo and the program's name
- Classes' saving goals and progress, fetched from Google Spreadsheets
- When relevant breaks start and end
- Public transport departure times, fetched from Västtrafik's API
- Latest post in a configured Google Classroom room, fetched from Google Classroom's API
- The current week, and the next week's school food, fetched from Skolmaten's API
